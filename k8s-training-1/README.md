# Workspace Summary

This repository is a Kubernetes training authoring workspace.

## What the automation is for

- **Instructions** define the writing rules for each content type, so labs, slides, quizzes, and solution guides stay consistent.
- **Agents** are specialized helpers for generating content, validating labs, checking consistency, reviewing accuracy, and writing commit messages.
- **Skills** bundle multi-step workflows for creating a new course or extending an existing one.
- **MCP servers** provide supporting capabilities such as file access, GitHub integration, and fetching external data.

## Available training-focused pieces

- Lab authoring standards
- Slide deck standards
- Speaker notes standards
- Self-study guide standards
- Assessment standards
- Solution guide standards
- Conventional commit guidance

- `training-day-generator`
